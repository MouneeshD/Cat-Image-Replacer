Cat Image Replacer

Cat Image Replacer is a lightweight Chrome Extension that adds a fun twist to web browsing by replacing every image on a webpage with random cat images. Built using HTML, CSS, JavaScript, and Manifest V3, the extension demonstrates how browser extensions can interact with webpage content in real time.

The extension includes features such as image detection, one-click image replacement, restoration of original images, and a modern popup interface that displays website information, image count, and operation status. This project provided hands-on experience with Chrome Extension APIs, DOM manipulation, message passing between scripts, and working with external image APIs.
